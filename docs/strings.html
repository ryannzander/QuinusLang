Strings & Builtins

String Type

The str type represents null-terminated character strings (C char*):

make shift s: str = "hello";
make shift greeting: str = "Hello, World!";

String Literals

Use double quotes. Escape sequences: \\ \" \n \t \r

"plain"
"with \"quotes\""
"line1\nline2"

String Concatenation

Use + to concatenate strings. The result is a newly allocated string:

make shift a: str = "Hello";
make shift b: str = " World";
make shift c: str = a + b;  // "Hello World"

String Interpolation

Use backtick strings with ${expr} to interpolate values. Works with print, write, and writeln:

make name: str = "world";
print(`Hello, ${name}!`);
make x: i32 = 42;
print(`x = ${x}`);

Escape backtick with \` inside interpolated strings.

I/O Builtins

BuiltinDescription
print(...)Print arguments to stdout, followed by newline
write(...)Print arguments to stdout, no newline
writeln(...)Print arguments to stdout with newline
read()Read an integer from stdin
print(42);
print("hello");
write("no newline ");
writeln("newline");
make shift x: i32 = read();

Assertion Builtins

BuiltinDescription
panic()Abort the program with a message
assert(cond)Abort if condition is false
assert(x > 0);
check (error) {
    panic();
}

Print Formatting

print, write, and writeln accept multiple arguments. Each is printed with an appropriate format (integer, float, string, etc.) based on its type.

File I/O (stdlib fs)

For reading and writing files, use the fs module:

bring "fs";

make f = fs.open_file("file.txt", "r");
make content = fs.read_all(f);
fs.close(f);

See Modules for the full standard library.
