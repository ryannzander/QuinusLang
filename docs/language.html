Language Basics

File Extension

QuinusLang source files use the .q extension.

Variables

Variables are declared with make (immutable) or make shift (mutable):

make x: i32 = 42;        // Immutable — cannot be reassigned
make shift y: i32 = 10;  // Mutable — can be reassigned

The type can be omitted if the compiler can infer it from the initializer:

make x = 42;           // Inferred as int
make shift s = "hi";   // Inferred as str

Identifiers

Identifiers must start with a letter or underscore, followed by letters, digits, or underscores:

foo
_bar
x42
myVariable

Comments

Single-line comments start with //:

// This is a comment
make x = 42;  // Inline comment

Statements and Semicolons

Statements end with a semicolon:

make shift x: i32 = 1;
x = x + 1;
print(x);
