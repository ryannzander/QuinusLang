# CLI Reference

| Command | Description |
|---------|-------------|
| `quinus build` | Compile to executable |
| `quinus build --release` | Optimized build |
| `quinus build --emit-c` | Emit C only |
| `quinus run` | Build and run |
| `quinus init` | Create new package |
| `quinus fmt` | Format source |
| `quinus watch` | Rebuild on changes |
| `quinus repl` | Interactive REPL |
