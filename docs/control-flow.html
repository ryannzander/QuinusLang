Control Flow

Conditionals: check and otherwise

Use check for if and otherwise for else:

check (x > 0) {
    print("positive");
}
otherwise {
    print("non-positive");
}

The condition must be a boolean expression in parentheses. The blocks use curly braces.

While Loop: loopwhile

make shift i: i32 = 0;
loopwhile (i < 10) {
    print(i);
    i = i + 1;
}

For Loop

C-style for loop with init, condition, and step:

for (make shift i: i32 = 0; i < 10; i = i + 1) {
    print(i);
}

Any of the three parts can be omitted (use semicolons to skip):

for (; cond; ) { ... }   // While-like
for (init; ; step) { ... }  // Infinite with manual break

For-Each Loop: foreach

Iterate over an array:

foreach item in arr {
    print(item);
}

item is a new variable in each iteration, bound to the current element.

Break and Continue

Use stop to break out of a loop and skip to continue to the next iteration:

loopwhile (true) {
    check (done) {
        stop;
    }
    check (skip_this) {
        skip;
    }
    // ...
}

Logical Operators

Short-circuit evaluation for && and ||:

check (x > 0 && x < 100) {
    // ...
}

check (err == 0 || fallback) {
    // ...
}
