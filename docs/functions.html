Functions

Defining Functions

Functions are declared with craft, followed by the name, parameters, optional return type, and body:

craft add(a: i32, b: i32) -> i32 {
    send a + b;
}

If the function returns nothing (void), the return type can be omitted or specified as void:

craft greet() -> void {
    print("Hello!");
    send;
}

craft no_return() {
    send;  // Implicit void
}

Parameters

Parameters are comma-separated, each with a name and type:

craft foo(x: i32, y: f64, name: str) -> void {
    // ...
    send;
}

Parameters are immutable by default. There is no variadic parameter support; use arrays or overloads for variable argument counts.

Returning Values

Use send to return. With no expression, it returns void. With an expression, it returns that value:

craft get_zero() -> i32 {
    send 0;
}

craft get_sum(a: i32, b: i32) -> i32 {
    send a + b;
}

Calling Functions

make shift result: i32 = add(1, 2);
print(add(10, 20));
greet();

Entry Point

Every executable program must define a main function with signature craft main() -> void:

craft main() -> void {
    print("Program started");
    send;
}

Recursion

Functions can call themselves:

craft factorial(n: i32) -> i32 {
    check (n <= 1) {
        send 1;
    }
    send n * factorial(n - 1);
}

Forward Declarations

Functions must be defined before use in the same file. For mutual recursion or cross-file calls, define the function in the order it's needed or use modules.
