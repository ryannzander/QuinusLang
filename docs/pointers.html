Pointers

Pointer Type: link

A pointer to type T is written link T:

make shift p: link i32;
make shift s: link str;

Address-of: mark

Get the address of a variable with mark:

make shift x: i32 = 42;
make shift p: link i32 = mark x;

p now holds the address of x.

Dereference: reach

Read or write through a pointer with reach:

make shift value: i32 = reach p;   // Read
reach p = 99;                       // Write

Complete Example

craft main() -> void {
    make shift x: i32 = 42;
    make shift p: link i32 = mark x;
    print(reach p);   // 42
    reach p = 100;
    print(x);         // 100
    send;
}

Pointer Arithmetic

Pointer arithmetic is not directly exposed in safe QuinusLang. Use array indexing or hazard blocks with inline assembly for low-level pointer manipulation.

Null Pointers

There is no built-in null literal. Use 0 or a cast for null pointer values when interfacing with C. Dereferencing null is undefined behavior.

C Mapping

link T compiles to T* in C. mark x becomes &x, and reach p becomes *p.
