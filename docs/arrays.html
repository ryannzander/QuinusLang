Arrays

Fixed-Size Arrays

Declare with [T; N] — N elements of type T:

make shift arr: [i32; 5] = { 1, 2, 3, 4, 5 };

The initializer must have exactly N elements (or fewer, with the rest zero-initialized depending on implementation).

Array Initializers

Use curly braces with comma-separated values:

make shift nums: [i32; 3] = { 10, 20, 30 };
make shift zeros: [i32; 5] = { 0, 0, 0, 0, 0 };

Indexing

Access elements with [index]:

make shift first: i32 = arr[0];
make shift last: i32 = arr[4];
arr[2] = 99;

Indices are zero-based. Bounds checking is not performed by default.

Length: len

Get the number of elements with the len builtin:

make shift n: usize = len(arr);
print(n);  // 5

Slices

Create a view into a subrange with [start..end]:

arr[1..4]   // Elements at indices 1, 2, 3
arr[..3]    // From start to index 3 (exclusive)
arr[2..]    // From index 2 to end

A slice produces a pointer to the first element of the range. The end index is exclusive.

Foreach Over Arrays

foreach item in arr {
    print(item);
}

C Mapping

[i32; 5] becomes int32_t arr[5] in C. Arrays are stack-allocated. Slices compile to pointer arithmetic.
