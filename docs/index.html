QuinusLang

A systems programming language with assembly-level control and readable syntax. QuinusLang compiles to C, which is then compiled to native executables by your system compiler (GCC, Clang, or MSVC).

Philosophy

  • Explicit control — Expensive or dangerous operations are visible in code
  • Readability — Low-level code should be easy to read and reason about
  • Assembly-level power — Inline assembly, pointers, zero-cost abstractions
  • Safety by design — Unsafe operations require explicit hazard blocks
  • Zero hidden runtime — Suitable for kernels, bootloaders, firmware, and embedded systems

Quick Example

craft main() -> void {
    make shift x: i32 = 42;
    print(x);
    print("Hello, QuinusLang!");
    send;
}
