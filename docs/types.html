Pointer Types

Use link T for a pointer to type T:

make shift p: link i32 = mark x;
make shift s: link str = mark my_string;

See Pointers for details.

Array Types

Fixed-size: [T; N] — array of N elements of type T

make shift arr: [i32; 5] = { 1, 2, 3, 4, 5 };

Unsized: [T] — used for slice results and generic array references

See Arrays for details.

Named Types

Struct, enum, and union names become types:

form Point { x: i32, y: i32 }
make shift p: Point = ...;

Type Inference

Variable types can often be inferred from the initializer:

make x = 42;        // i32 or int
make f = 3.14;      // f64
make b = true;      // bool
make s = "hi";      // str

Type Compatibility

Some implicit conversions are allowed (e.g., int to i32, int to f64). Array initializers must match the declared element type or be assignable.
